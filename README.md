# Rota Defteri Web

İstanbul mikro gezi rotalarını yönetmek için hazırlanmış React tabanlı web uygulaması.

## Özellikler

- Rota ekleme
- Rota listeleme
- Rota güncelleme
- Rota silme
- Kategori filtresi ve arama
- LocalStorage ile kalıcı kayıt
- Responsive arayüz

## Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Pure CSS
- LocalStorage

## Kurulum

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Proje Yapısı

```text
src/
  components/
  data/
  interfaces/
  pages/
  utils/
```
